# Momo Snake Lab

Momo Snake is a tiny GitHub-native, agent-native Snake MVP with Silicon Valley AI startup taste.

Live target:

```txt
https://mineh810.github.io/momo-snake-lab/
```

## Plane separation

- **Control plane**: PRD, taste guide, specs, review gates, and GitHub Actions delivery policy.
- **Agent plane**: Next.js implementation, game engine, local tests, Playwright checks, and deploy artifacts.

## Commands

```bash
npm install
npm run dev
npm run typecheck
npm run test
npm run build
npm run e2e
```

## MVP scope

- Playable Snake game
- WASD / arrow-key controls
- Score, high score, speed, status
- Local high-score persistence
- Run log and local heuristic Agent Assist panel
- GitHub Pages deployment through GitHub Actions

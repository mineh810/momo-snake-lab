# Agent Plane

The agent plane implements scoped work from the control plane.

## Responsibilities

- Implement game logic
- Implement UI
- Add tests
- Run local validation
- Produce screenshots
- Open reviewable PRs
- Avoid broad, unrequested architecture changes

## Current MVP implementation

- Next.js static export
- TypeScript game engine
- Tailwind UI
- LocalStorage high score
- Vitest unit tests
- Playwright smoke test and screenshot

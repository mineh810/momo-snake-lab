# Review Guide

## L0

Docs, copy, small polish. CI green is enough.

## L1

Small logic or UI changes. Agent review plus tests.

## L2

Gameplay, taste, architecture, deployment. Human review required.

## L3

Security, auth, Actions permissions, dependencies, production settings. Human approval required before implementation.

## MVP v1 gate

MVP v1 is accepted when the GitHub Pages URL is live, the game is playable, and screenshots show the intended AI-startup dashboard taste.

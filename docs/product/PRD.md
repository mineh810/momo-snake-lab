# Momo Snake PRD

## Vision

Momo Snake is a small browser Snake game presented as a premium AI-startup simulation dashboard.

The purpose is not just the game. The purpose is validating a GitHub-native, agent-native software workflow where a human product lead controls direction and agents execute scoped implementation tasks.

## MVP promise

A user can open the GitHub Pages URL, start a Snake run, control movement, score points, see telemetry, and feel that the product has polished AI-startup taste.

## Non-goals

- No backend
- No login
- No online leaderboard
- No real LLM API
- No payment
- No multiplayer

## Success criteria

- Web URL is live on GitHub Pages
- Game is playable
- UI has premium dark dashboard taste
- High score persists locally
- Unit tests cover core rules
- Playwright screenshot verifies visual rendering
- Control plane and agent plane are documented separately

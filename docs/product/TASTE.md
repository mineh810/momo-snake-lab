# Taste Guide

Momo Snake should look like a tiny interactive demo inside an AI infrastructure dashboard.

## References

- Linear
- Vercel
- Cursor
- Raycast
- Perplexity
- OpenAI dashboard

## Rules

- Dark mode first
- Thin borders
- Glass-like surfaces
- Muted gray text hierarchy
- Electric green / mint as snake accent
- Purple / indigo for agent assist
- Precise typography
- Calm motion
- No childish arcade styling
- No noisy rainbow gradients
